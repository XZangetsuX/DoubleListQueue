En el paquete APP se encuentra la clase "main" donde para 
probar o correr los metodos de Queue se debera de descomentar
las lineas de codigo.

Los metodos de Queue se encunetran en la clase DoubleListQueue
en el paquete doubleLinkedList ahi podra encontrar los metodos de Queue.

La imprecion se reliza con un metodos no sobre escrito llamado list que 
utiliza el metodo iterator(); para poder recorrer y mostrar la lista del 
primero al ultimo.

El metodo FrontOf se quedo en blanco ya que como no vimos prioridades 
no se pudo realizar (justificado).

Los metodos manejan exepciones tales como cuando esta vacia y cuando esta llena, 
estas exepciones saldran cuando se quiera agregar un 6 valores  ala lista ya que
solo se permite 5 definido en la variable size y cuando se quiera utilizar los metodos
front(),search() o deQueue() con la lista sin valores.
